================================================================================
ENCRYPTION KEY WORKFLOW
================================================================================

UPLOAD:
  1. Select file
  2. Generate AES-256 key
  3. Encrypt in browser
  4. Upload encrypted file
  5. Display key to user
  6. User SAVES key (Critical!)

DECRYPT:
  1. Go to "My Files"
  2. Click "Decrypt" on file
  3. Paste encryption key
  4. Key validated
  5. File decrypted in browser
  6. File downloads
  7. Activity logged

Server NEVER sees unencrypted data!
